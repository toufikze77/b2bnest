# Project Memory

## Core
- Email delivery exclusively uses Gmail SMTP via Google Workspace (notifications@b2bnest.online). No third-party email services (e.g., Resend).
- Multi-tenant architecture: users configure their own integrations, credentials stored per-user.
- AI system uses gpt-4o-mini; credit limits apply per plan and deduct before calls.
- DO NOT re-add: Business Directory, Marketplace, Phone Support.
- DO NOT re-add integrations for Trello, Notion, Slack, or NotePro email sync.
